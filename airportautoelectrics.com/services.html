Tytan security systems

Analogue alarms: Tytan DS300R PLIP

The Tytan DS300R is a high quality car alarm for vehicles with a central lock remote control. This alarm can also be controlled by Tytan remote transmitters. No need for an additional remote control makes this car alarm very popular among customers. Failure-free and long-term operation is guaranteed by specialized software for every car make.

Mobile App

Tytan GPS

Dedicated mobile application for Android and iOS allows to operate Tytan GPS DS511 and Tytan GPS DS512. Friendly interface allows to send lock/unlock commands to the vehicle, request GPS position and status of car, allows to control external heaters and has a build-in parking meter content(get on Google play ,App Store).

GSM/GPRS

Tytan GPS

Real-time tracking device. GSM/CPRS device designed for tracking vehicles and cargo or supervising technical objects in real time. It is intended for passenger, construction, money transport and agricultural vehicles, buses, trucks etc. DS520 communicates with server via GSM network GPS data(position, speed) and current parameters and events registered in the monitored object.
